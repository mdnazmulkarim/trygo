Initialize a git repo :

1. go inside the repo directory
Command :
git init

2. Add files:
git add filename

3. Commit:
git commit -m "initial commit"

4. Init to online :
git remote add origin https://github.com/mdnazmulkarim/trygo.git

5. Push to online:
git push origin master

6. Push subsequent :
git push origin master

7. Status:
git status

8. Adding all files:
git add .

9: git remove from next commit:
git rm filename
